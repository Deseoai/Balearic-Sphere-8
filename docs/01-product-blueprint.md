# Produkt Blueprint (V1)

## Positionierung

Kernsatz:

> Ein kuratiertes, KI-gestuetztes Zugangsnetzwerk fuer Mallorca, in dem Relevanz, Vertrauen und Chancen freigeschaltet werden.

Wirkung:

- Kein klassisches Community-Produkt
- Zugang statt Massen-Registrierung
- Status durch Beitrag, Trust und Fit

## Sprachstil fuer Landing und Onboarding

Primar:

- Request Access
- Apply for Entry
- See if you qualify
- Not open. Curated.

Vermeiden:

- Join now
- Register free
- Open community

## Membership-Mechanik

- Public: Landing + Bewerbung
- Application: strukturierte Angaben (Offer, Seek, Why Join)
- AI Pre-Scoring: Relevanz/Fit/Trust-Signale
- Admin Review: Freigabe fuer Premium Layer
- Access Level: explorer -> curated -> verified -> insider -> private_circle_eligible

## Credit Economy (V1)

Einstieg:

- +100 Welcome
- +25 Profil komplett

Spending (MVP):

- Profil/Node Unlock
- AI Anfrage
- Intro Request

Earning (MVP):

- Qualitaetsbeitrag im Forum
- bestaetigte Intro-Hilfe
- verifizierte Aktivitaet

## Kernmodule im MVP

1. Landing + Access Funnel
2. Bewerbungsprozess
3. Access/Review-Layer
4. Credits Ledger
5. AI Concierge basic
6. Marketplace lite
7. Forum lite
