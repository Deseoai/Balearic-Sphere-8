# MVP Roadmap (6 Wochen)

## Woche 1: Foundation

- Repo + Deploy-Basis
- Postgres/Redis
- Landing + Access Form
- API Grundendpunkte

## Woche 2: Access Engine

- Application Pipeline
- AI Pre-Scoring v1
- Admin Review Basis
- Access Level Assignment

## Woche 3: Credits

- Wallet + Transactions
- Spend/Reward Regeln
- Anti-Abuse Regeln (Limits, Cooldowns)

## Woche 4: Marketplace + Forum Lite

- Listing CRUD
- Forum Rooms + Posts
- Basic moderation flags

## Woche 5: AI Concierge Basic

- Chat endpoint
- profile-aware prompt context
- credit charging per request

## Woche 6: Hardening

- Webhook security tests
- observability (errors, logs, alerts)
- onboarding copy polish
- closed alpha launch (20-50 Member)

## KPI fuer Closed Alpha

- Bewerbungen/Woche
- Acceptance Rate
- Time-to-First-Value
- Credits earned vs spent
- Intro request completion
